# airess glow

A collection of four beautiful, carefully curated VS Code themes from the airess design system.

## Themes

- **lumen airess** - Light theme with lilac purple accents ✨
- **airess bloom** - Light theme with rosy pink accents 🌸
- **midnight airess** - Dark theme with magenta noir accents 🌙
- **velvet airess** - Dark theme with deep indigo accents 💜

- ## Features

- Apply UI fonts option for a polished look
- Carefully selected color palettes for reduced eye strain
- Support for bracket pair colorization
- Optimized cursor styling and behavior
- Inlay hints and code lens styling

## Installation

1. Open VS Code
2. Go to Extensions (Ctrl+Shift+X / Cmd+Shift+X)
3. Search for "airess glow"
4. Click Install

## Usage

After installation, select your preferred theme:

1. Open Command Palette (Ctrl+Shift+P / Cmd+Shift+P)
2. Type "Color Theme"
3. Choose from:
   - lumen airess
   - airess bloom
   - midnight airess
   - velvet airess

### Apply UI Fonts

This extension includes a command to apply UI font preferences (uses the system sans-serif stack by default):

1. Open Command Palette (Ctrl+Shift+P / Cmd+Shift+P)
2. Search for "airess glow: Apply UI fonts"
3. The extension optionally integrates with the [Custom UI Style](https://marketplace.visualstudio.com/items?itemName=subframe7536.custom-ui-style) extension for additional UI overrides

## Requirements

- VS Code 1.74.0 or higher
- [Custom UI Style](https://marketplace.visualstudio.com/items?itemName=subframe7536.custom-ui-style) extension (optional, for custom UI fonts)
